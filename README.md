Data Analytics Portfolio – Dashboard Projects

This repository contains my Data Analytics dashboard projects created using Microsoft Power BI and Microsoft Excel.

Projects Included:

1. Call Center Dashboard (Power BI)
   - Analysis of call volume, agent performance, resolved vs unresolved calls, and customer satisfaction.

2. Sales / Business KPI Dashboard (Power BI)
   - Interactive dashboard tracking business performance and key metrics.

3. Financial Analytics Dashboard (Power BI)
   - Revenue, profit, and cost insights using Power BI visuals.

4. Swiggy Sales Dashboard (Excel)
   - Excel dashboard analyzing food delivery orders, revenue, and customer trends.
   - Built using Pivot Tables, charts, slicers, and interactive filters.

Tools & Skills Used:
• Microsoft Power BI Desktop  
• Microsoft Excel  
• Power Query  
• Data Modeling  
• DAX Measures  
• Pivot Tables & Charts
